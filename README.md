# Claude Code Toolkit

Skills, agents, and commands for Claude Code / Clawdbot.

## Skills

### Humanizer
Remove signs of AI-generated writing from text. Based on Wikipedia's comprehensive "Signs of AI writing" guide.

### Research
Evidence-based research workflow applying meta-research best practices. Implements MECE decomposition, hypothesis-driven investigation, iterative refinement, source quality tracking, and confidence indicators.

## Installation

Install skills via [ClawdHub](https://clawdhub.com) or copy directly into your Clawdbot skills directory.

## Structure

```
skills/          # AgentSkills for Clawdbot
agents/          # Custom agent configurations (coming soon)
commands/        # CLI commands and scripts (coming soon)
```
